## Overview
A simple server-side mod that stops bats from spawning.  
Perfect for players or servers who want cleaner caves and fewer unnecessary mobs.

## Commands
All commands require OP level 2 permission.

**View Current Status:**
```
/easeon batbegone
```

**Enable Bat Blocking:**
```
/easeon batbegone on
```

**Disable Bat Blocking:**
```
/easeon batbegone off
```

## Configuration 
```json
{
  "enabled": true,
  "requiredOpLevel": 2 // Requires a server restart to take effect.
}
```
`config/easeon/easeon.ss.batbegone.json`
